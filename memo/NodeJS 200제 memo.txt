🍘 Terminal

npm init -y

nodemon
: Automatic restarting of application.

🍘 package.json

"scripts": {
    "test__<예제번호>": "nodemon --exec node example/**/<실행할 파일명>"
 },

: <실행할 파일명>을 **/<번호>*.js 로 작성하여 설정해둠.  

실행 시 'npm run test__예제번호'로 확인

🍘 console.log

: console이라는 Class와 .log() 라는 method를 사용하여 출력한다. 

console.log(), console.error(), console.warn() 등이 주로 쓰인다.

🍘 Formatting

:%d
: 숫자, 소수점을 나타낸다.

:%s
: 문자, 문자열을 나타낸다.

기본적으로 console.log는 문자열을 반환한다. Java에 printf( )와 유사하다.
